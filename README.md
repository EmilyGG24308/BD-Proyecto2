**Como levantar el proyecto**
1. Tener Docker instalado y corriendo
2. Clona el repo

- git clone https://github.com/EmilyGG24308/BD-Proyecto2.git
- cd BD-Proyecto2
4. Copia el .env
- cp .env.example .env
6. Levanta todo
- docker compose up --build
8. Abre: http://localhost:8080
